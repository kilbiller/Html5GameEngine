#Html5GameEngine

A simple HTML5 game framework made for fun & experience

## Features

* ECS (Entity/Component/System Architecture)
* AssetManager
* SpriteSheet Animations
* Collision detection & response
* StateManager
* Variable Time Step

* stats-js for the fps meter
